# Claude IDE Site

Landing page for Claude IDE, a terminal-based development workspace.

## Overview

A static website serving as the landing page for Claude IDE. Includes a product story, demo videos, keyboard shortcut references, and install/uninstall scripts. Hosted via GitHub Pages with a custom CNAME.

## Contents

- `index.html` -- Main landing page
- `script.js` / `style.css` -- Frontend logic and styling
- `keyboard.html` -- Keyboard shortcut reference
- `get/` / `uninstall/` -- Install and uninstall scripts
- `Claude_Prompt.mp4`, `ctrl+p.mov` -- Demo videos

## Getting Started

Visit the hosted site or open `index.html` locally in a browser.
